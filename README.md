# Interactive Quiz Application

A modern, interactive quiz application built with React, Vite, and the Open Trivia Database API.

## Features

- **API Integration**: Fetches quiz questions from Open Trivia Database API
- **Modern ES6+ JavaScript**: Uses arrow functions, destructuring, async/await, spread operator, and more
- **Multiple Categories**: Choose from various quiz categories
- **Difficulty Levels**: Select Easy, Medium, or Hard questions
- **Real-time Scoring**: Track your score as you progress
- **Responsive Design**: Works seamlessly on all devices
- **Interactive UI**: Beautiful animations and feedback

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

## Technologies Used

- React 18
- Vite
- Open Trivia Database API
- ES6+ JavaScript Features
- CSS3 with Animations
